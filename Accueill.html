BIENVENUE SUR MA PAGE D'APPRENTISSAGE EN DEVELOPPEMENT WEB

Definition des concepts
1. Internet L'Internet est un réseau mondial d'ordinateurs interconnectés qui permet l'échange de données. Il constitue l'infrastructure qui permet à des services comme le Web, l’e-mail, le transfert de fichiers, etc.

2. Web (ou World Wide Web) Le Web est un service qui fonctionne sur Internet. Il permet d'accéder à des pages et des sites web via des navigateurs (comme Chrome, Firefox). Le Web utilise principalement le protocole HTTP pour la communication.

3. Git Git est un logiciel de gestion de versions. Il permet de suivre les modifications apportées à des fichiers, en particulier dans le développement de logiciels, et de collaborer efficacement entre développeurs.

4. GitHub GitHub est une plateforme en ligne qui utilise Git. Elle permet d’héberger des projets, de partager du code, de collaborer avec d’autres développeurs, et de gérer les versions et les contributions.

5. Site Web Un site web est un ensemble de pages web liées entre elles, regroupées sous un même nom de domaine (ex : www.example.com). Il peut contenir du texte, des images, des vidéos, etc.

6. Page Web Une page web est un document individuel consultable sur le Web, généralement écrit en HTML, et qui peut faire partie d’un site web. Exemple : la page d’accueil d’un site.

7. HTML (HyperText Markup Language) HTML est le langage de balisage utilisé pour structurer le contenu d'une page web. Il sert à définir les titres, paragraphes, liens, images, etc.

8. CSS (Cascading Style Sheets) CSS est un langage qui permet de styliser les pages HTML. Il sert à définir les couleurs, les polices, les marges, l’agencement des éléments, etc.
